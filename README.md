# 医療・介護 業務支援統合システム「Medivisor（メディバイザー）」 🏥

## 1. Project Overview
医療・介護現場の業務（マニュアル閲覧・有給申請・勤怠管理）を統合管理するプラットフォームです。

## 2. Tech Stack
- **Backend**: Java / Spring Boot 3
- **Frontend**: React / Vite
- **Database**: MySQL 8.0
- **Infrastructure**: Docker / Docker Compose

## 3. Quick Start
```bash
docker compose up --build
```
